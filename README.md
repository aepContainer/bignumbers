# BigNumbers - Library for Performing Operations with Large Numbers

**BigNumbers**, is a C library designed to perform mathematical operations on large numbers and much more...  

## Supported Operations:

  - Addition
  - Multiplication
  - Extraction
  - Division
  - Exponentiation
  - Factorialization
  - Modulo
